Semester Courses

  • Web Technology

    This course aims to provide students with knowledge and required expertise in client-side and server-side scripting for web application development. The course introduces various methods and technologies used in developing web applications including HTML5, CSS, DHTML using jQuery, JAVASRIPT, PHP, and MySQL to develop a complete web site. The course introduces these technologies from the viewpoint of application development and hands-on. The course will conclude with a final project building a basic client website using all of the skills acquired in the course. It is expected that the skills and knowledge gained will help students to develop a web 3.0 applications.

    Lecturer: Dr. Alimatu-Saadia Yussiff

  • Research Methods

    This course focuses on developing the skills needed to plan, conduct, and report on research in the Computer Science, Information Technology, and Information Systems domains. This course will cover techniques applicable to each of the steps of a research project, including formulating research questions, theory building, data analysis (using both qualitative and quantitative methods), building evidence, assessing validity, and publishing. The approaches that would be used in the delivery of the course will cognitively engage students and also ensure the learning progress of all students irrespective of their gender, academic capability, and ethnicity.

    Lecturer:Dr. Alimatu-Saadia Yussiff

  • Advanced Calculus II

    Advanced Calculus delves deeper into the study of functions, limits, derivatives, and integrals, extending concepts from elementary calculus. Topics include multivariable calculus, sequences and series, vector calculus, and advanced techniques for solving differential equations. The course explores the rigorous mathematical foundations of calculus, emphasizing theoretical understanding and practical applications in various fields such as physics, engineering, and economics.

    Lecturer: Dr. Emily Johnson

  • Entrepreneurship

    Entrepreneurship introduces principles and practices of creating, managing, and growing a business venture. It covers topics such as identifying opportunities, market research, business planning, financial management, marketing strategies, and risk assessment. The course emphasizes the development of entrepreneurial mindset, creativity, innovation, and leadership skills necessary for starting and sustaining successful businesses. Students learn how to navigate challenges, capitalize on opportunities, and create value in diverse industries and contexts

    Lecturer: Prof. Michael Brown

  • Networking Fundamentals

    Networking explores the principles, protocols, and technologies underlying computer networks. It covers topics such as network architectures, communication protocols, data transmission, network security, and network management. The course delves into both theoretical concepts and practical applications, including local area networks (LANs), wide area networks (WANs), the Internet, and emerging technologies like cloud computing and Internet of Things (IoT). Students learn to design, implement, and troubleshoot network infrastructures, preparing them for roles in network administration, cybersecurity, and telecommunications.

    Lecturer: Dr. Emmanuel D. Tettey

  • Human-Computer Interaction

    Human-Computer Interaction (HCI) explores the design, evaluation, and usability of interactive systems. It covers topics such as user interface design principles, interaction design methodologies, usability testing, and user experience (UX) research. The course integrates knowledge from psychology, design, and computer science to understand how humans interact with technology and how to create intuitive, efficient, and enjoyable user experiences. Students learn to analyze user needs, design user interfaces, and evaluate system usability, equipping them with skills for designing user-friendly software applications, websites, and digital products.

    Lecturer: Prof. David White

  • Programming Analysis with Java

    Programming Analysis with Java focuses on the application of Java programming language in problem-solving and software development. It covers topics such as object-oriented programming concepts, data structures, algorithms, and software design principles. The course emphasizes the analysis of complex problems and the design of efficient solutions using Java programming constructs. Students learn to write well-structured, modular, and maintainable code, and to apply debugging and testing techniques to ensure program correctness and reliability. Additionally, it explores advanced Java features and libraries to develop robust and scalable software applications for various domains, including web development, mobile apps, and enterprise systems

    Lecturer: Dr. Elliot Attipoe
